# Snowflake 学習リポジトリ

Snowflake の資格対策、実践ハンズオン、横断ノートを、教材の種類ごとに整理したリポジトリです。

## 構成

| ディレクトリ | 用途 |
| --- | --- |
| `apps/learning-hub/` | Associate / Core を横断する学習ノート Web アプリ |
| `certifications/` | SnowPro Associate・Core の試験対策教材と問題演習アプリ |
| `hands-on/` | Snowflake Badge、Cortex Search、dbt の実践教材 |
| `shared/` | 資格横断の学習マップと共通トピック索引 |

## 学習の入口

- GitHub Pages では、ルートページから学習ノートと各資格の問題演習を開けます。
- 資格試験対策は `certifications/snowpro-associate/` と `certifications/snowpro-core/` を使います。
- 実践学習は `hands-on/snowflake-badges/`、`hands-on/cortex-search/`、`hands-on/dbt/` を使います。
- 共通テーマは `shared/00_学習マップ.md` と `shared/01_共通トピック索引.md` からたどれます。

## GitHub Pages のURL互換性

公開元は新しい構成に合わせています。既存の `/app/`、`/SnowPro_Core/quiz_app/`、`/SnowPro_Associate/quiz_app/` のURLも、互換用コピーとして引き続き利用できます。
