# Snowflake 学習マップ

Associate と Core の学習領域を横断して確認するためのマップです。

## 資格別の位置づけ

| 資格 | 主なレベル感 | 学習の中心 |
| --- | --- | --- |
| SnowPro Associate | 入門から基礎操作 | Snowflake の基本概念、UI、SQL 基礎、ロード、セキュリティ、Cortex など |
| SnowPro Core | 実務基礎から運用理解 | アーキテクチャ、アクセス制御、パフォーマンス、ロード、保護、共有など |

## 学習順序の目安

1. `shared/01_共通トピック索引.md` で全体像を確認する。
2. Associate の `まとめ/` で基本概念を固める。
3. Core の `確認問題/` で試験頻出の観点を確認する。
4. 間違えた領域を Associate/Core 双方のノートで見直す。
