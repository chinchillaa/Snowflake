## 問題8: Reader Account

Reader Accountについて説明せよ。

### あなたの回答

```
Reader Accountとは:


Reader Accountの用途:


作成者: [Provider / Consumer]

費用負担者: [Provider / Consumer]

制限事項:
  1. 
  2. 
```

### 正解・解説

<details>
<summary>クリックして確認</summary>

**Reader Account:** ProviderがSnowflakeアカウントを持たない組織のために作成する、読み取り専用のSnowflakeアカウント

**用途:** Snowflakeアカウントを持たない外部パートナーや顧客とデータを共有

**作成者:** **Provider**

**費用負担者:** **Provider**（ストレージとコンピュート両方）

**制限事項:**
1. 読み取り専用（データ作成・変更不可）
2. 作成したProviderのデータのみアクセス可能
3. 他のProviderからのShareは利用不可

</details>

```
CURRENT_ACCOUNT()関数の用途:


実装例（概念）:

  SELECT * FROM my_table
  WHERE tenant_id = [                          ];
```

### 正解・解説

<details>
<summary>クリックして確認</summary>

**方法:** セキュアビューとコンテキスト関数を組み合わせる

**CURRENT_ACCOUNT():** 現在接続しているアカウントの識別子を返す

**実装例:**
```sql
CREATE SECURE VIEW shared_view AS
SELECT * FROM my_table
WHERE tenant_id = CURRENT_ACCOUNT();
```

これにより、消費者ごとに自分のデータのみが表示される。
