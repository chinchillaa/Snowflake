# SnowPro Associate: Platform Certification (SOL-C01) 試験概要

> 最終更新: 2026-03-23

---

## 試験基本情報

| 項目 | 内容 |
|------|------|
| 試験コード | SOL-C01 |
| 試験名 | SnowPro Associate: Platform Certification |
| 問題数 | 65問 |
| 試験時間 | 85分 |
| 合格スコア | 750点以上（0〜1000点） |
| 受験料 | $100 USD（インド: $80 USD） |
| 試験方式 | オンライン（Pearson VUE、非監督） |
| 有効期限 | 2年間 |
| 開始日 | 2025年2月 |

---

## 出題ドメインと配点

| ドメイン | 配点 | 主なトピック |
|---------|:----:|--------------|
| **Data Management and Loading** | 25% | COPY INTO、ステージ、ファイル形式、Snowpipe |
| **Snowflake Accounts and Security** | 20% | アカウント構造、認証、暗号化 |
| **Snowflake User Interface and Notebooks** | 15% | Snowsight、Notebooks、ワークシート |
| **Data Access and Roles** | 15% | RBAC、ロール階層、権限管理 |
| **Snowflake Cortex AI/ML Functions** | 15% | Cortex LLM関数、AI機能 |
| **Data Protection and Sharing** | 10% | Time Travel、Fail-safe、Secure Data Sharing |

---

## 受験要件

### 前提条件
- **なし**（誰でも受験可能）

### 推奨経験
- Snowflakeでの実務経験: **3ヶ月以上**
- SQLの基本的な知識
- クラウドデータウェアハウスの概念理解

---

## 学習期間の目安

| 経験レベル | 推奨学習期間 |
|-----------|:------------:|
| Snowflake初心者 | 6〜8週間 |
| Snowflake経験者（3ヶ月+） | 4〜6週間 |
| 他のDWH経験あり | 3〜4週間 |

## 試験当日のポイント

### 形式
- 多肢選択式（単一選択・複数選択）
