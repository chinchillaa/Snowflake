# 📓 SELECT StarとLIMIT

## 📓 SELECT * とLIMITについて学ぶ

**SELECT * 文**（セレクトスター）は、テーブルの**すべてのカラム**を一度に取得する方法です。アスタリスク（*）を使うため、カラムを1つずつ列挙する必要がありません。

すべてのカラムが欲しいが、すべての行は不要な場合は、**LIMIT** を付けて実行します。LIMITは少数の行だけを取得するようにします。数百万行ある場合に、全行を取得してコンピューティングリソースを無駄にしないで済みます。

```sql
SELECT *
FROM ROOT_DEPTH 
LIMIT 1;
```

## 📓 SQLが初めて？ヘルプが必要？以下のコードサンプルを参考にしてください

```sql
insert into root_depth (root_depth_id, root_depth_code
     , root_depth_name, unit_of_measure
     , range_min, range_max)  
values
     (5,'X','short','in',66,77)
     ,(8,'Y','tall','cm',98,99)
;
```

行を削除するには：

```sql
delete from root_depth
where root_depth_id = 9;
```

特定の行のカラム値を変更するには：

```sql
update root_depth
set root_depth_id = 7
where root_depth_id = 9;
```

すべての行を削除してやり直すには：

```sql
truncate table root_depth;
```
